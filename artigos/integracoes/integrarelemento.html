Como integrar o Bulldesk com um formulário do Elementor

Caso você tenha apenas um formulário por página



Após criar a sua página utilizando o Elementor e adicionar o seu formulário, adicione um elemento "Shortcode" o mais para o final da página possível:



No shortcode, adicione o script de integração do Bulldesk, conforme o exemplo abaixo:
<script src="//static.bulldesk.com.br/forms.js"></script>
<script>
  new BulldeskFormIntegration({
    token: 'SEU_TOKEN_VAI_AQUI',
    identifier: 'ESCOLHA UM IDENTIFICADOR PARA ESSA CONVERSÃO',
    validate: true,
    map: {
        'form_fields[name]': 'name',
        'form_fields[email]': 'email'
    }
  });
</script>


Não esqueça de alterar SEU_TOKEN_VAI_AQUI pelo seu token do Bulldesk
Não esqueça de mapear os campos do seu formulário, utilizando o map presente no exemplo de script fornecido acima.
Se você tem dúvidas de onde pegar o seu token ou como mapear os campos do formulário, leia este artigo.

Prontinho, feito isso sua página já estará integrada

Caso você tenha mais de um formulário por página



Utilize esse passo apenas se na sua página tiver mais de um formulário

Insira os campos do seu formulário normalmente. Quantos campos quiser.



Feito isso, adicione um campo tipo Hidden e coloque o nome de identifier



Em seguida clique em Advanced em Default Value coloque o nome que você quer dar a este formulário.

Os leads que preencherem este formulário vão chegar no Bulldesk com o nome da conversão que você preencheu aqui. Dessa forma, você poderá saber de qual formulário o lead veio e criar automações com isso. Ex: Cabeçalho Home



Se você seguir os passos acima, a integração com o formulário foi realizada.

Agora você precisa adicionar o script para mapear os campos dentro do Bulldesk. Esse código é em encontrado na parte de configurações do Bulldesk. 

<script src="//static.bulldesk.com.br/forms.js"></script>
<script>
  new BulldeskFormIntegration({
    token: 'SEU_TOKEN_VAI_AQUI',
    validate: true,
    map: {
        'form_fields[identifier]': 'identifier',
        'form_fields[name]': 'name',
        'form_fields[email]': 'email'
    }
  });
</script>
