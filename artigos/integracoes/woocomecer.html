Como integrar o Salesflow com o Woocommerce

Para realizar a integração do Salesflow com o Woocommerce, acesse seu WordPress (wp-admin).

No menu esquerdo, selecione a opção Woocommerce > Configurações



Em seguida, clique na opção Avançado > Webhooks > Adicionar webhook





Preencha os seguintes campos:

Nome: Escolha um nome para esse webhook, esse nome não será encaminhado para o Salesflow, serve apenas para reconhecimento interno.

Status:
Ativado: (entregando);
Pausando: (não esta entregando);
Desabilitado: (não esta entregando por causa de falhas na entrega);

Tópico: Escolha quando os dados serão enviados para o Salesflow, esse será o gatilho do seu webhook. Exemplo: caso eu escolha a opção pedido criado, toda vez que for realizado um pedido de compra no meu site, os dados desse lead serão enviados para o Salesflow.

Utilizando a opção de tópico "Ação", você consegue inserir uma ação personalizada para que sirva de gatilho, para isso é necessário conhecer os códigos das ações utilizados pelo Woocommerce, e esse código deve ser inserido no "Evento da ação".

URL de entrega: insira a URL da API do Salesflow.

Para ações relacionadas ao cliente (exemplo: cliente criado), você deve utilizar seguinte URL: 
https://app.bulldesk.com.br/hook/woocommerce/TOKEN_PUBLICO/customer


Para ações relacionadas aos pedidos (exemplo: pedido criado), você deve utilizar seguinte URL: 
https://app.bulldesk.com.br/hook/woocommerce/TOKEN_PUBLICO/order


No caso de outras ações você pode utilizar a URL: 
https://api.bulldesk.com.br/conversion/TOKEN_PUBLICO


Não esqueça de substituir o TOKEN_PUBLICO pelo seu token, seu token pode ser encontrado aqui.

Segredo: A chave secreta é usada para gerar um hash do webhook entregue e é fornecido nos cabeçalhos da solicitação. Por padrão a chave de API customer secret do usuário será utilizado. (caso você não saiba o que inserir, utilize o mesmo nome colocado no título).
